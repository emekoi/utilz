# palette file spec
palettes can passed to pal as `palette-profile` where no `-profile` means to use the default palette. 
```toml
; the only required profile
[default]
foreground = 0xrrggbb
background = 0xrrggbb
cursor = 0xrrggbb
colors = [
  0xrrggbb, ; color 1
  0xrrggbb, ; color 2
  ...
  0xrrggbb, ; color 16
]

[light]
foreground = ...
background = ...
cursor = ...
colors = [
  ...
]

[dark]
foreground = ...
background = ...
cursor = ...
colors = [
  ...
]

[random-profile-name]
foreground = ...
background = ...
cursor = ...
colors = [
  ...
]
```
